Header¶

This is the root element (/) of an HDF5 file satisfying the ASDF definition.

Attributes¶

file_format¶

Description: The file format version.

Value: "0.0.2"

Details:

ATTRIBUTE "file_format" {
    DATATYPE  H5T_STRING {
        STRPAD H5T_STR_NULLPAD;
        CSET H5T_CSET_ASCII;
        CTYPE H5T_C_S1;
    }
    DATASPACE  SCALAR;
}

file_format_version¶

Description: The file format name.

Value: "ASDF"

Details:

ATTRIBUTE "file_format_version" {
    DATATYPE  H5T_STRING {
        STRPAD H5T_STR_NULLPAD;
        CSET H5T_CSET_ASCII;
        CTYPE H5T_C_S1;
    }
    DATASPACE  SCALAR;
}
